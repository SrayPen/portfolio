# 青甘小环线4天自驾游详细路程安排

## 第一天 (10月2日): 兰州 - 武威 - 张掖

### 1. 兰州 → 武威
- **出发地**: 兰州中心广场 (36.0613, 103.8341)
- **出发时间**: 06:30
- **路线**: G30高速公路
- **距离**: 约285公里
- **预计行驶时间**: 3小时
- **到达时间**: 09:30
- **目的地**: 武威雷台 (37.9283, 102.6388)

### 2. 武威游览
- **时间**: 09:30 - 11:30
- **景点**: 
  - 武威雷台
  - 西夏博物馆
- **用餐**: 11:30 - 12:30 在武威用午餐（推荐：武威牛肉面）

### 3. 武威 → 张掖
- **出发时间**: 12:30
- **路线**: G30高速公路
- **距离**: 约175公里
- **预计行驶时间**: 2小时
- **到达时间**: 14:30
- **目的地**: 张掖丹霞国家地质公园 (38.9414, 100.4487)

### 4. 张掖丹霞游览
- **时间**: 15:00 - 18:00
- **景点**: 七彩丹霞地貌
- **提示**: 建议在日落前到达，欣赏最美丹霞景色

### 5. 张掖住宿
- **时间**: 18:30
- **建议**: 如家酒店(张掖西大街店) (38.9261, 100.4595)

**总行驶距离**: 约460公里

## 第二天 (10月3日): 张掖 - 祁连 - 青海湖 - 西宁

- **出发时间**: 05:30
- **路线**: G227国道
- **距离**: 约180公里

- **到达时间**: 14:00
- **目的地**: 青海湖二郎剑景区 (36.5889, 100.4867)

### 4. 青海湖游览
- **时间**: 14:00 - 16:00
- **活动**: 
  - 参观二郎剑景区

### 5. 青海湖 → 西宁
- **出发时间**: 16:00
- **路线**: G109国道
- **距离**: 约150公里
- **预计行驶时间**: 2小时
- **到达时间**: 18:00
- **目的地**: 西宁市区，青海宾馆 (36.6166, 101.7770)

**总行驶距离**: 约630公里

## 总行程概览
- **总行驶距离**: 约1990公里
- **行程天数**: 4天
- **主要景点**: 张掖丹霞、祁连山、青海湖、拉卜楞寺、卓尼土司官寨等

## 注意事项
1. 所有时间和距离均为估计值，实际情况可能因交通状况而有所不同。
2. 高原地区请注意防范高原反应，准备相应药物。
3. 长时间驾驶请注意休息，确保行车安全。
4. 景点开放时间和门票价格可能会有变动，建议提前确认。
5. 国庆期间道路可能拥堵，请预留足够时间。
6. 请关注天气预报，准备适当的衣物和装备。
